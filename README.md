# Speech to Text App

A Python desktop application that converts spoken Romanian speech into text using microphone input.

Built with CustomTkinter and SpeechRecognition, the app allows users to record audio, display the transcribed text in a simple GUI, clear the text area, and save the result as a `.txt` file.

## Technologies
- Python
- CustomTkinter
- SpeechRecognition

## Features
- Record speech from microphone
- Convert speech to text
- Display text in the app
- Save transcription to file
- Clear text output
